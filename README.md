The following implementations need to be reviewed:

## Rules
The code should be fully runnable. **NOT:** 

    var sum = 0
    for i in 1...1000000000 {
		sum += i
	}
	print (sum)
	
**Instead do:**

    class ViewController: UIViewController {
        func viewDidLoad() {
        	var sum = 0
            for i in 1...1000000000 {
            	sum += i
            }
            print (sum)
        }
    }
