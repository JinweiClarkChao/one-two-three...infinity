The following implementations need to be reviewed:

+ [Mathematica](sum.m)

## Rules
The code should be fully runnable. (The following example is Swift) **NOT:** 

```swift
    var sum = 0
    for i in 1...1000000000 {
		sum += i
	}
	print (sum)
```

**Instead do:**

```swift
    class ViewController: UIViewController {
        func viewDidLoad() {
        	var sum = 0
            for i in 1...1000000000 {
            	sum += i
            }
            print (sum)
        }
    }
```
